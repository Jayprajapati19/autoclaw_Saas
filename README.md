# Autoclaw SaaS — AI Agent Platform

Autoclaw is a TypeScript-based AI agent platform that helps developers analyze, plan, and automate tasks on a local codebase through CLI or Telegram. It can research, generate plans, and apply code changes with user approval, ensuring safe and controlled automation.

### Features
- AI-powered code analysis and task automation
- CLI and Telegram bot support
- Research, planning, and execution workflows
- Approval-based code changes for safety
- Web search and scraping integration
- Built with TypeScript and Bun
